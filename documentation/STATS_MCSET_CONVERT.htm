Convert a Multiple Category Set to a Multiple Dichotomy Set

This procedure converts a multiple category set to a multiple dichotomy set representing the same information. Variables in an MC set are all coded the same way but hold different values.

You might want to do this in order to use the STATS CATEGORY ORDER extension command to set a specific category order for use with CTABLES or in order to tabulate individual values

Multiple Category Set Select a multiple category set for conversion.

• The source Variables control shows both multiple category and multiple dichotomy sets.

Prefix for Generated Variable Names Enter a prefix for the names of the dichotomous variables to be generated. The names will have the form
prefix_nn
where nn is a number

• Existing variables may be overwritten. If a string variable with one of these names exists, delete it before running this procedure.

Name for New Set Enter a name to be assigned to the new MD set.

Additional Features

This dialog generates syntax for the STATS MCSET CONVERT extension command. To display syntax help for this command, run the following syntax:

STATS MCSET CONVERT /HELP.

Requirements

This command requires the Python Essentials You can download these via the SPSS Community website at www.ibm.com/developerworks/spssdevcentral or from your installation materials.
